# Trackpoint-Config-OSX

Config Utility for ThinkPad TrackPoint Bluetooth Keyboards for OSX

## Support Models
- ThinkPad Compact Bluetooth Keyboard with TrackPoint (0B47189)
- ThinkPad TrackPoint Keyboard II

## Install
1. `make install`
2. `sudo tpkb`
